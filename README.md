# Career-Coaching-Guide
2026 Career Coaching guide as a mobile-optimized HTML app with compact design and practical content.
The design features:
Key Features:
Small fonts & 1% margins as requested for compact mobile viewing
6 major 2026 coaching themes including AI integration, hybrid work, skills-based pivoting, purpose-driven careers, wellness integration, and gig economy strategies
Complete 10-step guide on how to create a career coaching practice from scratch
Interactive accordion sections for expanded methodologies
Modern design with gradient backgrounds, color-coded sections, and responsive layout
Practical tools including pricing structures, tech stack recommendations, and success metrics
The guide covers everything from obtaining certifications and building your framework to client acquisition strategies and avoiding common pitfalls. Perfect for viewing on mobile devices with easy navigation and comprehensive information.

Practical Tutorials:

Step-by-Step Resume Review Tutorial - How to analyze and improve client resumes with before/after examples
LinkedIn Profile Optimization Guide - Section-by-section tutorial for maximizing client profiles
Mock Interview Script Templates - Ready-to-use scenarios for different industries/roles
Salary Negotiation Roleplay Guide - Scripts and frameworks coaches can use with clients
Personal Branding Workshop Template - Complete session outline for helping clients build their brand
Career Pivot Assessment Tutorial - How to evaluate transferable skills and new pathways

Client-Facing Resources:

30/60/90 Day Job Search Plan Templates - Structured plans clients can follow
Networking Email Templates - Cold outreach, follow-ups, informational interviews
Job Application Tracker - System for managing multiple applications
Interview Preparation Checklist - Pre-interview, during, and post-interview actions
